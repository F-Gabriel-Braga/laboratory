Comunicação entre componentes

É a forma como utilizamos os recursos do Angular para transmitir dados entre os componentes na aplicação.

Input

Entrada de dados de um componente pai para um componente filho. Utiliza-se o decorator @Input() para atribuir a uma propriedade do componente a entrada de dados do seu componente pai.

Output

Saida de dados de um componente filho para um componente pai. Aplica-se o decorator @Output() sobre um objeto do tipo EventEmitter que enviará os dados via evento.

Exemplo

Externo

{{ "Nome: " + pessoa.nome + " Idade: " + pessoa.idade }}

Template Variable

Template Variables são variáveis que o angular nos permite definir dentro dos elementos HTML do template do seu componente. A sintaxe de Template Variable é "#" e o nome que você definir.

{{ texto }}

ViewChild

View Child permite acessar dados, atributos e métodos de elementos ou componentes do template HTML. Utiliza-se o decorator @ViewChild para anotar a propriedade que receberá o elemento HTML ou o componente Angular.

Dado interno: {{ interno.propriedade }}

Campo possui o valor: {{ campo.value }}
